**IcedWave Cleaner** — это мощная, быстрая и абсолютно безопасная утилита для очистки операционной системы Windows от накопленного мусора, временных файлов, кэша приложений и игр. Написана на Python с использованием современного графического интерфейса **CustomTkinter** в темных тонах. **ВАЖНО! МЫ НЕ НЕСЕМ ОТВЕТСТВЕННОСТЬ ЗА ТО ЧТО ВЫ СЛУЧАЙНО НАЖАЛИ НА ОДНУ ИЗ ПАПОК, КОТОРУЮ НЕ ДОЛЖНЫ БЫЛИ И УДАЛИЛИ!!**

Программа оптимизирует пространство на диске, не затрагивая важные системные данные и личные файлы пользователя.

## ✨ Основные фишки и возможности

* 🚀 **Умное сканирование в реальном времени** — подсчет мусора работает в отдельных фоновых потоках, благодаря чему интерфейс никогда не зависает.
* 🛡️ **Абсолютная безопасность** — встроенный умный фильтр пропускает критические системные папки Windows (System32, WinSxS), защищая систему от случайных поломок.
* 🎮 **Глубокая очистка кэша** — находит и удаляет гигабайты скрытого мусора от лаунчеров (Steam, Epic Games, Riot Client) и тяжелых игр.
* 📦 **Поддержка софта для разработчиков и дизайнеров** — безопасная очистка кэша JetBrains (PyCharm), VS Code, Android Studio, Gradle, Adobe Premiere, Photoshop, Figma, Blender и др.
* 🏎️ **Режим максимальной производительности** — переключение схемы питания Windows на скрытый высокопроизводительный режим одной кнопкой.
* 📂 **Удаление пустых папок** — рекурсивный поиск и зачистка пустых директорий на выбранных дисках для наведения полного порядка.
* 🔄 **Автоматическое обновление** — встроенная система проверки версий прямо через GitHub Releases с автоматическим бесшовным апдейтом.
* 🎨 **Красивый UI** — стильный и лаконичный Dark Theme интерфейс, вдохновленный современным дизайном Windows 11.
